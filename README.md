# cn-ssl-hub

本仓库用于归档和发布多个独立的 HTML 页面。

## 项目简介

cn-ssl-hub 是一个静态页面集合仓库，主要存放与 SSL/TLS 相关的信息页面、工具页面或说明页面。所有页面均为独立 HTML 文件，不依赖外部数据库或后端服务。

仓库本身不针对任何特定域名、网站或服务，仅作为页面存档与发布用途。

## 目录说明

```
.
├── index.html          # 首页 / 导航页
├── pages/              # 存放各个独立 HTML 页面
│   ├── example.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，文件名即页面标识。
- `assets/` 目录存放页面共用的样式、脚本和图片资源。

## 页面归档说明

- 所有页面均为纯前端静态 HTML，可直接在浏览器中打开。
- 页面内容不涉及任何具体域名、网址或商业推广。
- 页面可能包含技术说明、配置参考或通用工具界面。
- 页面之间相互独立，无跳转依赖关系。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 以更新或添加页面。
- 新增页面请放入 `pages/` 目录，并保持文件名简洁。
- 如需引用公共资源，请优先使用 `assets/` 下的本地文件。
- 请勿在页面中插入任何形式的广告、追踪代码或诱导点击内容。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等服务部署后访问。
